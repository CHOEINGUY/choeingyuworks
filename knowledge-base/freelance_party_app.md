# 외주 2 - 부산 승인제 파티 업체 참가자 관리 시스템 (2025.09)

## 한 줄 요약
네이버 창업자 카페에 글을 올린 후 연락이 온 부산 승인제 파티 업체. AppSheet로 참가자 관리를 개선하고, 개인화 초대장 + QR 출석 체크 + 입금 자동 확인까지 제안해 구현한 프로젝트.

## 시작

역학조사 시스템 완료 후 할 일이 없을 때, 네이버 창업자 카페에 "자동화 서비스를 할 수 있다"는 글을 올렸다. 부산의 승인제 파티 업체에서 연락이 왔다.

기존에는 구글 시트 + Apps Script로 관리하고 있었는데, 참가자 취소/변경 처리 시 로직이 복잡해지는 문제가 있었다. 최인규가 먼저 AppSheet로 바꾸는 것을 제안하며 영업. 경험 목적이라 금액을 거의 안 받겠다고 했다.

## 구현 내용

**참가자 관리**: AppSheet로 남녀 구분, 날짜별 분류. 슬라이스로 날짜별 뷰 구성 → 취소/변경 시에도 테이블 구조가 유지돼 로직이 깔끔해짐.

**문자 발송 자동화**: Flask 서버 + 웹훅으로 문자 대행사 연동. 참가 확정 시 자동 발송.

**개인화 초대장 페이지**: React로 개발. 참가자별 QR 코드 삽입. Supabase 연결 (실시간 처리 필요해서 선택). QR 코드 스캔 시 자동 출석 체크 → 즉시 환영 페이지로 전환 → 파티에서 사용하는 웹 프로그램으로 바로 이동하는 흐름. 이 흐름 전체는 최인규가 먼저 제안했고 클라이언트가 수락했다.

**입금 확인 자동화**: 통장 이체 내역을 끌어와 이름/시간 매칭으로 자동 처리. 오류 가능성 있어서 안전 장치 추가.

## 결과

지금도 잘 사용 중.
