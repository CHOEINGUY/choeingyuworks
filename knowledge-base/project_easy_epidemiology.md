# 역학조사 분석 시스템 (Easy Epidemiology) - 기술 상세

> 배경, 동기, 개요는 커버레터 참고.

## 첫 시도의 구체적 실패

2024년 연말, Vue 3로 시작. AI와 상의해서 바이브 코딩으로 개발했는데, 개발할수록 에러가 쌓이고 뭐가 문제인지 알 수 없는 상태가 됐다. AI에게 프로젝트 구조를 설명했더니 "스파게티 코드가 너무 심하다. 컴포넌트 파일 하나에 코드가 3000줄이 넘는다"는 피드백이 왔다.

웹 개발 경험이 없다 보니 프로젝트 관리 자체를 몰랐다. Git도 몰라서 압축 파일로 백업하고 있었다. "백업1, 백업2, 백업최종..."

전면 리셋 후 AI와 충분히 논의해서 바이브 코딩에 적합한 폴더 구조·컴포넌트 분리 원칙을 먼저 확립하고 재개발. 리셋 이후가 오히려 더 수월했다.

## 개발 타임라인

- 2024년 연말: Vue 3로 첫 시도 → 실패, 전면 리셋
- 2025년 6월: 초안 완성
- 2025년 7월: 광주/전남 보건소 담당자 대상 발표 및 배포
- 현재: 사용 중

## 개발·검증 구조

전문의 선생님이 공공보건의료지원단(광주 소재) 자문을 맡고, 그 선생님을 통해 최인규에게 개발을 의뢰한 형태. 최인규는 공식 사업 인원이 아니었지만 개발과 피드백 대응 전담. 6월 초안 완성 후 공공보건의료지원단 담당자들 + 보건소 담당자들과 지속적인 미팅을 통해 보고서 형식, 차트 자동화, 추가 기능 등을 계속 개선.
