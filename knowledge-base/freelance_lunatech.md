# 외주 1 - 루나테크 방충망 업체 견적서/세금계산서 자동화 (2024.09)

## 시작 배경

당근마켓 알바 구인 글을 보고 먼저 계약서 작성 앱을 샘플로 만들어 영업했다. 사장님이 "계약서는 필요 없고, 업체 고객 관리 + 견적서 발행 앱을 만들어 달라"고 역제안. 그 요청을 받아 AppSheet로 개발했다.

## 기술 구현 디테일

클라이언트 추가 요청: 전자세금계산서 발행 연동.

**문제**: AppSheet의 텍스트 템플릿이 JSON 형태가 아니라 세금계산서 대행사 API에 직접 연동하기 불가.

**해결**: Google Cloud에 Flask 서버를 열고 AppSheet → Flask(웹훅) → 세금계산서 대행사로 데이터를 중계하는 구조로 설계.

## 처음 해본 것들

이 프로젝트에서 처음 써본 것들이 한꺼번에 쏟아졌다:
- AppSheet 웹훅 (외부 서비스 연동 자체가 처음)
- Flask 서버 직접 구축
- Google Cloud 배포
- 외부 API(세금계산서 대행사) 연동

외부 서비스를 한 번도 연결해본 적 없어서 막막했지만, 바이브 코딩으로 계속 시도하다 결국 해냈다. 현재까지도 사장님이 실사용 중이며 간간이 추가 요청이 오고 있다.
