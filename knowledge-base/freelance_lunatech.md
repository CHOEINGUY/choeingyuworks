# 외주 1 - 루나테크 방충망 업체 견적서/세금계산서 자동화 (2024.09)

> 당근마켓 알바로 연결된 과정은 커버레터 참고.

## 기술 구현 디테일

클라이언트가 필요한 것: 업체 거래용 견적서 작성 + 전자세금계산서 발행 자동화.

**문제**: AppSheet의 텍스트 템플릿이 JSON 형태가 아니라 세금계산서 대행사 API에 직접 연동하기 불가.

**해결**: Google Cloud에 Flask 서버를 열고 AppSheet → Flask → 세금계산서 대행사로 데이터를 중계하는 구조로 설계. 이때 처음 Flask 서버를 직접 구축해본 경험.

이후에도 간간이 연락이 오며 추가 요청을 받고 있다.
