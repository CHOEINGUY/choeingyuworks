# 외주 3 - 인천 CPR 출강 업체 강사 관리 통합 시스템 (2026.01~)

## 한 줄 요약
크몽에서 연결된 인천 CPR(심폐소생술) 출강 업체의 강사 15명 관리 시스템. AppSheet + Google Chat + 바이브 코딩 웹앱 + 이수증 조회 페이지까지 통합적으로 구축한 프로젝트.

## 시작

크몽에 서비스를 올린 지 한 달쯤 지나 인천 CPR 출강 업체에서 연락. 강사 15명을 구글 시트 + Apps Script로 관리하다 너무 불편해서 통합 관리 시스템이 필요하다고 했다.

## 기술 스택

- AppSheet (강사 관리, 강의 신청/배정, 현장 보고, 정산)
- Google Workspace + Google Chat (채팅방 자동 생성, 알림)
- Google Apps Script (자동화, 보고서 생성, 블로그 포스팅 자동화)
- 바이브 코딩 웹앱 (강의 일정 매트릭스 대시보드)
- HTML/JS + Supabase (CPR 이수증 조회 웹페이지)

## AppSheet 구성

**강사 기능**:
- 강의 신청: 모집 중인 강의 목록 확인 후 신청
- 내 강의 목록: 관리자가 승인 확정한 강의 확인
- 오늘 강의: 당일 강의 확인 + 현장 도착 후 체크인(출근) - GPS 좌표와 시간 기록
- 결과 보고: 강의 종료 후 대표사진 1장 + 추가사진 + 메모 제출. 팀 강의는 팀원 중 1명만 먼저 제출하면 나머지는 버튼이 사라지는 선착순 방식.
- 월별 정산 관리: 월별 강의 참석 내역 집계

**관리자 기능**:
- 강의 등록/배정 승인
- 전체 교육 현황 모니터링
- 강사 인원 관리

**개인정보 보호**: 결과보고 사진 제출 시 Cloudinary를 통해 자동으로 얼굴 모자이크 처리 후 원본은 백업 폴더에 보관.

## Google Chat 연동

- 강의 배정 확정 시 해당 강사들만 포함된 전용 채팅방(스페이스) 자동 생성
- 전날/당일 아침 예비 알림 자동 발송
- 교육 결과 보고 완료 시 알림

Google Workspace 도입 이유: AppSheet 유료 계정으로 15명 인원 전원 사용하고 Google Chat 연동을 하려면 Workspace가 필요했다.

## 강의 일정 매트릭스 대시보드

AppSheet로는 구현이 안 되는 매트릭스 형태의 대시보드 (강사 × 날짜로 모든 강의 일정을 한눈에 조작)를 바이브 코딩으로 웹앱으로 직접 개발해 연동.

## 블로그 포스팅 자동화

Apps Script (BlogReportGenerator.gs, PhotoMosaic.gs)로 결과보고 사진을 활용해 블로그 포스팅을 자동 생성.

## 이수증 조회 웹페이지 (CPR Certificate)

- **구성**: HTML/CSS/JS (프레임워크 없음) + Supabase(PostgreSQL)
- **흐름**: 관리자가 구글 시트에 이수자 데이터 입력 → 시트 상단 버튼 클릭 시 Apps Script가 Supabase로 업로드 → 완료 데이터는 보관용 시트로 자동 이동
- **사용자 조회**: 성명 + 생년월일 입력 → 이수증 랜더링 → PDF 다운로드 / 인쇄
- **발급 이력**: PDF 최초 다운로드 시 발급 일시가 Supabase에 기록되고 구글 시트에도 양방향 동기화
- **보안**: RLS(Row Level Security)로 성명+생년월일 일치 시에만 데이터 반환
- **호스팅**: 정적 HTML 파일 → Netlify/Vercel 배포. 기존 아임웹 홈페이지에서 버튼 링크로 연결.
